# Binary ↔ Text Converter

A lightweight, accessible static web page that converts text to 8-bit binary and binary back to text.

## Features
- Text ↔ Binary conversion
- Dark/light mode toggle
- Copy/download actions
- LED animation feedback and optional sound

## Usage
1. Open `index.html` in your browser.
2. Enter text or binary and click the desired button.
3. Copy or download the result as `.txt`.
